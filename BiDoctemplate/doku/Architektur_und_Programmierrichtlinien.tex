\subsection{Formatierungen}
Es gibt zwei Alternativen hierfür:
\begin{itemize}
	\item \textbf{Mittels SSIS:} Die Konvertierungen werden mittels SISS Komponenten durchgeführt.
	\item \textbf{Mittels SQL:} Die Daten werden als Text in eine DB integriert und dann danach mittels SQL Konvertierungen in das Zielformat überführt
\end{itemize}
